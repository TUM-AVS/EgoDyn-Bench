## Technical Details

### nuScenes Coordinate System

- **x-axis:** Forward (vehicle front)
- **y-axis:** Left
- **z-axis:** Up
- **Yaw:** Rotation around z-axis
- **Positive yaw rotation:** Counter-clockwise from above = LEFT turn

### Derivation Methods

- **Yaw:** Extracted from ego_pose quaternion → Euler angles
- **Yaw-rate:** Numerical derivative of unwrapped yaw: `(yaw[i+1] - yaw[i-1]) / (2*dt)`
- **Speed:** `sqrt(dx² + dy²) / dt` from position deltas
- **Acceleration:** Numerical derivative of speed
